# Notification Service in Nest

## Description

Notification service in [Nest](https://github.com/nestjs/nest) framework using TypeScript, Prisma and Jest.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
